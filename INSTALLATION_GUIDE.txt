===============================================================
UNIVERSITY RESULT MANAGEMENT SYSTEM - Quick Installation Guide
===============================================================

STEP 1: Prerequisites
---------------------
✓ XAMPP/WAMP/MAMP installed
✓ PHP 7.4+ 
✓ MySQL 5.7+
✓ Web browser

STEP 2: Setup
-------------
1. Copy project folder to web server root:
   - XAMPP: C:\xampp\htdocs\
   - WAMP: C:\wamp\www\
   - MAMP: /Applications/MAMP/htdocs/

2. Start Apache and MySQL servers

3. Import database:
   - Open phpMyAdmin (http://localhost/phpmyadmin)
   - Click "Import" tab
   - Select file: database/schema.sql
   - Click "Go"

4. Configure database (if needed):
   - Edit config/database.php
   - Update DB_USER and DB_PASS if different

STEP 3: Access Application
--------------------------
Open browser: http://localhost/university-management-system/

===============================================================
TROUBLESHOOTING
===============================================================

Q: Database connection error?
A: Check MySQL is running and credentials in config/database.php

Q: Login not working?
A: Ensure database is imported correctly with sample users

Q: Pages showing blank?
A: Check PHP error logs in XAMPP/WAMP/MAMP logs folder

Q: Styling issues?
A: Clear browser cache (Ctrl+Shift+Delete)
